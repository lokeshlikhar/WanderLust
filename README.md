# 🌍 WanderLust – Airbnb Inspired Listing Platform

A full-stack web application inspired by Airbnb, built using **Node.js, Express.js, MongoDB, and EJS**. Users can explore property listings, create and manage their own listings, upload images, leave reviews, and search destinations through a clean and responsive interface.

## 🔗 Live Preview

**🌐 Live Website:** https://wanderlust-assl.onrender.com

---

## ✨ Key Features

* 🏡 Browse and explore property listings
* ➕ Create, edit, and delete listings
* 🔐 Secure user authentication with Passport.js
* ⭐ Add and delete reviews with ratings
* 📍 Search listings by destination
* 🗺️ Interactive location maps using Mapbox
* 📷 Image uploads with Cloudinary
* 👤 Authorization so only listing/review owners can modify their content
* 📱 Fully responsive user interface
* ⚡ Server-side validation and custom error handling

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3, Bootstrap 5, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Authentication:** Passport.js, Express Session, bcrypt
* **Cloud Storage:** Cloudinary, Multer
* **Maps & Geocoding:** Mapbox API
* **Validation:** Joi
* **Deployment:** Render
